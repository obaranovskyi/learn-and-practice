# Exercises: Nominal That-Clauses

Identify function (subject, object, complement): 1. That she won surprised us. 2. I believe that he's honest. 3. The problem is that we're late. 4. The fact that he left is strange. 5. I'm glad that you came.

**Answers:** 1. subject 2. object 3. complement 4. after noun 5. after adjective
