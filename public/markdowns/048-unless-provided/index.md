# Unless, As long as, Provided that

## Unless = If not
- I'll go **unless** it rains. (= if it doesn't rain)
- **Unless** you hurry, you'll be late.

## As long as / So long as = Only if
- I'll help **as long as** you pay me.
- You can go **so long as** you're back by 10.

## Provided (that) / Providing (that) = Only if (formal)
- I'll agree **provided that** you sign the contract.
- **Providing** you work hard, you'll succeed.

## On condition that (formal)
- I'll tell you **on condition that** you keep it secret.
