# Second Conditional

Form: **If + past simple, would + base verb**

## Use: Unreal/unlikely present or future

### Examples
- If I **had** money, I **would buy** a car. (I don't have money)
- If she **were** here, she **would help**. (She's not here)
- What **would** you **do** if you **won** the lottery?

## Were vs Was
- Formal: If I **were** you... (subjunctive)
- Informal: If I **was** you...
