# Common Phrasal Verbs

Essential phrasal verbs by base verb.

## GET
- **get up**: rise from bed
- **get on/off**: board/exit (bus, train)
- **get in/out**: enter/exit (car, taxi)
- **get over**: recover from
- **get along with**: have good relationship

## TAKE
- **take off**: remove / depart
- **take up**: start (hobby)
- **take after**: resemble (family)
- **take on**: accept (work, employee)

## PUT
- **put on**: wear / add weight
- **put off**: postpone / discourage
- **put up with**: tolerate
- **put away**: store

## COME
- **come across**: find accidentally
- **come up with**: think of
- **come back**: return

## GO
- **go on**: continue / happen
- **go off**: explode / sound (alarm)
- **go out**: leave / extinguish

## LOOK
- **look for**: search
- **look after**: care for
- **look forward to**: anticipate
- **look up**: search (dictionary)
- **look into**: investigate

## TURN
- **turn on/off**: activate/deactivate
- **turn up/down**: increase/decrease (volume)
- **turn out**: result
