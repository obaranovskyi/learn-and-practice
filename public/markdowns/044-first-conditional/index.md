# First Conditional

Form: **If + present simple, will + base verb**

## Use: Real/likely future situations

### Examples
- If it **rains** tomorrow, I **will stay** home.
- If she **passes** the exam, she **will celebrate**.
- I **won't go** if you **don't come**.

## Variations
- If + present, **might/may/could** (less certain)
- If + present, **can** (ability/permission)
- If + present, **imperative** (instructions)

Examples:
- If you're tired, you **can** leave early.
- If it rains, **take** an umbrella.
