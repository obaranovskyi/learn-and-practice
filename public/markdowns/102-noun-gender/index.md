# Noun Gender

English has natural gender (based on sex) rather than grammatical gender.

## Gender-Specific Nouns

| Male | Female |
|------|--------|
| man | woman |
| boy | girl |
| father | mother |
| brother | sister |
| uncle | aunt |
| nephew | niece |
| king | queen |
| prince | princess |
| actor | actress |
| waiter | waitress |
| hero | heroine |

## Gender-Neutral Trends

Modern English prefers gender-neutral terms:
- *actor* (for all genders)
- *flight attendant* (not stewardess)
- *firefighter* (not fireman)
- *police officer* (not policeman)
- *chairperson/chair* (not chairman)
- *server* (not waiter/waitress)

## Pronouns for Things

- Ships, countries: traditionally *she* (now often *it*)
- Animals (unknown sex): *it*
- Pets (known to speaker): *he/she*
