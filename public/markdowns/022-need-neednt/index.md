# Need to / Needn't

## Need to (regular verb)
- I **need to** go.
- She **needs to** study.
- Do you **need to** leave?

## Needn't (modal - British)
- You **needn't** worry. (= don't need to)
- **Needn't** we stay? (rare)

## Needn't vs Don't need to
- Same meaning: no necessity
- **Needn't** = more British
- **Don't need to** = common everywhere
