# Exercises: None/Neither

Choose the correct verb:

1. None of the water (is/are) clean.
2. Neither answer (is/are) correct.
3. None of them (has/have) arrived.
4. Neither the cat nor the dogs (was/were) outside.
5. Either solution (is/are) acceptable.

**Answers:** 1. is 2. is 3. has/have (both ok) 4. were 5. is
