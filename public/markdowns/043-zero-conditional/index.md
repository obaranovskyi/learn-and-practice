# Zero Conditional

Form: **If + present simple, present simple**

## Use: General truths, facts, always true

### Examples
- If you heat water to 100Â°C, it **boils**.
- If it rains, the grass **gets** wet.
- Plants **die** if they don't get water.

## Also with "when"
- **When** you press this button, the light comes on.

## Notes
- Both clauses use present simple
- "If" can be replaced by "when"
- Describes cause and effect
