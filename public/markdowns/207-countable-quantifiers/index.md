# Countable Quantifiers

These quantifiers are used **only with countable nouns**.

## Many

Large number:
- ***Many** people came.*
- *How **many** books do you have?*
- *Not **many** students passed.*

## Few / A few

Small number:
- ***A few** people know.* (some, positive)
- ***Few** people know.* (not many, negative)
- *Very **few** understand this.*

## Several

Moderate number (3-7):
- ***Several** cars were parked.*
- *I've visited **several** countries.*

## A number of

Plural verb:
- ***A number of** students **are** absent.*

## Common Countable Quantifiers

| Quantifier | Meaning |
|------------|---------|
| many | large number |
| few | small number (negative) |
| a few | small number (positive) |
| several | moderate number |
| a number of | some (plural verb) |
| numerous | many (formal) |
| a couple of | two or a few |

## Remember

These ONLY work with countable plurals:
- ✓ *many books, few people, several options*
- ✗ *many water, few information, several advice*

For uncountable, use: *much, little, a little*
