# Exercises: Both Countable & Uncountable

Use a lot of, plenty of, or enough:

1. We have ___ time. Don't worry.
2. There are ___ options to choose from.
3. Is there ___ food for everyone?
4. ___ students are attending the event.
5. We don't have ___ chairs for all guests.

**Answers:** 1. plenty of/enough 2. a lot of/plenty of 3. enough 4. A lot of 5. enough
