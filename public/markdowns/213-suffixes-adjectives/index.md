# Suffixes for Adjectives

Adjective suffixes form adjectives from nouns/verbs.

## Common Suffixes

- **-ful**: care → careful, beauty → beautiful (full of)
- **-less**: care → careless, hope → hopeless (without)
- **-able/-ible**: enjoy → enjoyable, access → accessible (can be)
- **-ous/-ious**: danger → dangerous, space → spacious
- **-ive**: act → active, create → creative
- **-al**: nation → national, music → musical
- **-ic**: artist → artistic, science → scientific
- **-y**: sun → sunny, rain → rainy
- **-ly**: friend → friendly, love → lovely
- **-ish**: child → childish, self → selfish

## Opposites

- **-ful** vs **-less**: helpful / helpless
- careful / careless
- hopeful / hopeless
