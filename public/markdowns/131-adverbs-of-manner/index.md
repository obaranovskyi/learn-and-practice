# Adverbs of Manner

Adverbs of manner describe **how** an action is performed.

## Formation

Most are formed by adding **-ly** to adjectives:
- *quick → quickly*
- *careful → carefully*
- *beautiful → beautifully*
- *easy → easily*

## Spelling Rules

- **-y → -ily**: *happy → happily*
- **-le → -ly**: *gentle → gently*
- **-ic → -ically**: *automatic → automatically*
- **-ful → -fully**: *careful → carefully*

## Irregular Forms

| Adjective | Adverb |
|-----------|--------|
| good | well |
| fast | fast |
| hard | hard |
| late | late |
| early | early |
| straight | straight |

## Position

Usually **after the verb** or **after the object**:
- *She speaks **quietly**.* (after verb)
- *He read the letter **carefully**.* (after object)
- *They did the work **well**.*

## Common Adverbs of Manner

- *slowly, quickly, carefully, badly*
- *loudly, quietly, softly, clearly*
- *easily, suddenly, properly, politely*
- *fluently, perfectly, correctly*
