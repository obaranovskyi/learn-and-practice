# Attributive vs Predicative Adjectives

## Attributive Position

Before the noun:
- *a **tall** man*
- *the **red** car*
- *an **interesting** book*

## Predicative Position

After linking verbs (be, seem, become, etc.):
- *The man is **tall**.*
- *The car looks **red**.*
- *The book seems **interesting**.*

## Only Attributive (before noun)

Some adjectives can ONLY go before nouns:
- **main**: *the main reason* (✗ The reason is main)
- **chief**: *the chief problem*
- **principal**: *the principal cause*
- **only**: *the only answer*
- **mere**: *a mere child*
- **sheer**: *sheer luck*
- **utter**: *utter nonsense*
- **former/latter**: *the former president*

## Only Predicative (after verb)

Some adjectives can ONLY go after verbs:
- **afraid**: *She is afraid.* (✗ an afraid girl)
- **alive**: *The fish is alive.*
- **asleep/awake**: *He is asleep.*
- **alone**: *She is alone.*
- **well** (healthy): *He feels well.*
- **ill**: *She is ill.*
- **aware**: *I'm aware of it.*

## Different Meaning

| Attributive | Predicative |
|------------|-------------|
| *an old friend* (longtime) | *My friend is old.* (aged) |
| *a late train* (scheduled) | *The train is late.* (delayed) |
