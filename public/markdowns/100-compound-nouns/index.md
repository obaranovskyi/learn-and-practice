# Compound Nouns

Compound nouns combine two or more words to create a new meaning.

## Forms

### One Word
- *bedroom, toothbrush, sunlight*
- *football, newspaper, homework*

### Two Words
- *bus stop, coffee shop, ice cream*
- *car park, post office*

### Hyphenated
- *mother-in-law, passer-by*
- *check-in, grown-up*

## Patterns

| Pattern | Examples |
|---------|----------|
| noun + noun | bus stop, coffee cup |
| adjective + noun | blackboard, software |
| verb + noun | swimming pool, washing machine |
| noun + verb | sunrise, haircut |
| verb + preposition | checkout, breakdown |
| preposition + noun | underground, overcoat |

## Stress Pattern

Usually stress the **first** word:
- **TOOTH**brush, not tooth**BRUSH**
- **BUS** stop, not bus **STOP**
- **BLACK**board, not black**BOARD**

## Plural Formation

Pluralize the main noun:
- *toothbrushes* (one word → add -s at end)
- *bus stops* (two words → last word)
- *mothers-in-law* (hyphenated → main noun)
- *passers-by* (main noun = passer)
