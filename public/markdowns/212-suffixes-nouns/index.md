# Suffixes for Nouns

Noun suffixes turn verbs/adjectives into nouns.

## From Verbs

- **-tion/-sion**: act → action, decide → decision
- **-ment**: develop → development, achieve → achievement
- **-ance/-ence**: perform → performance, exist → existence
- **-er/-or**: teach → teacher, act → actor
- **-al**: arrive → arrival, refuse → refusal

## From Adjectives

- **-ness**: happy → happiness, kind → kindness
- **-ity**: able → ability, real → reality
- **-dom**: free → freedom, bore → boredom
- **-th**: grow → growth, warm → warmth

## Abstract Nouns

- **-ship**: friend → friendship, leader → leadership
- **-hood**: child → childhood, neighbor → neighborhood

## Person Nouns

- **-ist**: art → artist, science → scientist
- **-ian**: music → musician, magic → magician
