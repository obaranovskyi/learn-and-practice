# There is / There are

"There" introduces existence; verb agrees with the **real subject** that follows.

## Basic Rule

- **There is** + singular: *There **is** a book on the table.*
- **There are** + plural: *There **are** books on the table.*

## All Tenses

| Tense | Singular | Plural |
|-------|----------|--------|
| Present | there is | there are |
| Past | there was | there were |
| Present Perfect | there has been | there have been |
| Future | there will be | there will be |

## With Uncountable Nouns

Always singular:
- *There **is** some water.*
- *There **is** a lot of traffic.*
- *There **was** no information.*

## Contracted Forms

- *There's* = There is (singular)
- Informal: *There's* also used with plural in speech

## Lists

Traditionally, agree with **first item**:
- *There **is** a book and two pens.*
- *There **are** two pens and a book.*

Modern usage: many use "there are" for any plural list.

## Common Mistakes

- ✗ *There is many people here.*
- ✓ *There **are** many people here.*

- ✗ *There's a lot of problems.*
- ✓ *There **are** a lot of problems.*

## Questions

- ***Is** there a problem?*
- ***Are** there any questions?*
- ***Was** there anyone home?*
