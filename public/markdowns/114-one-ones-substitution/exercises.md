# Exercises: One / Ones

Fill in one, ones, or leave blank:

1. I like this hat, but I prefer that ___.
2. Which car? The red ___.
3. These books are cheaper than those ___.
4. I'll have two ___, please. (ordering apples)
5. Is this your pen or mine ___?

**Answers:** 1. one 2. one 3. ones 4. (blank) 5. (blank - don't use after possessive pronouns)
