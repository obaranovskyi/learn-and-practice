# Articles with Geographical Names

## Use "the"

| Category | Examples |
|----------|----------|
| Oceans/Seas | the Pacific Ocean, the Red Sea |
| Rivers | the Mississippi, the Danube |
| Canals | the Suez Canal, the Panama Canal |
| Deserts | the Sahara Desert, the Kalahari |
| Mountain ranges | the Rocky Mountains, the Andes |
| Island groups | the Maldives, the Canary Islands |
| Regions | the Middle East, the South |

## No Article

| Category | Examples |
|----------|----------|
| Continents | Africa, Europe, Antarctica |
| Countries (most) | Canada, Germany, China |
| States/Provinces | Texas, Ontario, Bavaria |
| Cities | Paris, Sydney, Mumbai |
| Single islands | Bali, Sicily, Crete |
| Single mountains | K2, Kilimanjaro |
| Lakes | Lake Michigan, Lake Geneva |
| Parks | Central Park, Hyde Park |

## Special Cases

- *the Congo* (country or river - context determines)
- *the Hague* (exception - always "the")
