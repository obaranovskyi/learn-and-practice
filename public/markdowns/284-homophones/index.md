# Homophones

Words that sound the same but have different meanings/spellings.

## Common Pairs

| Word 1 | Word 2 | Meanings |
|--------|--------|----------|
| their | there | possession / place |
| to | too / two | preposition / also / number |
| hear | here | listen / place |
| know | no | understand / negative |
| write | right | put on paper / correct |
| see | sea | vision / ocean |
| meet | meat | encounter / food |
| wear | where | clothing / place |
| week | weak | 7 days / not strong |
| be | bee | verb / insect |
| by | buy / bye | near / purchase / goodbye |

## More Pairs

- **piece/peace** - part / calm
- **flour/flower** - baking / plant
- **weather/whether** - climate / if
- **whole/hole** - complete / opening
- **break/brake** - damage / stop
- **wait/weight** - stay / heaviness
- **ate/eight** - past of eat / number
- **won/one** - victory / number
- **son/sun** - male child / star
- **pair/pear** - two / fruit

## Tricky Sets

- **they're/their/there**
- **you're/your**
- **it's/its**
- **who's/whose**
- **we're/were/where**

## Tips

- Check meaning in context
- Spell-check won't catch these
- Practice common pairs
- Read the sentence aloud
