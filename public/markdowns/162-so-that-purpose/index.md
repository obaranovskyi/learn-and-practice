# So that, In order that (Purpose)

These expressions show the purpose or intention of an action.

## So that

**Meaning**: in order to achieve something
- *I'll leave early **so that** I won't be late.*
- *Speak louder **so that** everyone can hear.*
- *She saved money **so that** she could travel.*

Often followed by **can/could/will/would**:
- *...so that I **can** see better.*
- *...so that she **could** understand.*

## In order that

More formal than "so that":
- *We arrived early **in order that** we might get good seats.*
- *He spoke slowly **in order that** everyone could understand.*

## So as (not) to + Infinitive

- *I left early **so as to** avoid traffic.*
- *She spoke quietly **so as not to** wake the baby.*

## In order (not) to + Infinitive

- *I study hard **in order to** pass.*
- *He saved money **in order to** buy a car.*
- *Be quiet **in order not to** disturb them.*

## Comparison

| Expression | Structure | Example |
|------------|-----------|---------|
| so that | + clause | so that I can see |
| in order that | + clause (formal) | in order that we might |
| so as to | + infinitive | so as to arrive early |
| in order to | + infinitive | in order to succeed |
| to | + infinitive (simple) | to learn English |
