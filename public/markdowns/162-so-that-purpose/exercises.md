# Exercises: Purpose

Complete with so that, in order to, or so as to:

1. She studied hard ___ pass the exam.
2. Speak louder ___ everyone can hear.
3. I left early ___ avoid the rush.
4. He exercises ___ he can stay healthy.
5. Walk quietly ___ not wake the baby.

**Answers:** 1. in order to / to 2. so that 3. so as to / in order to 4. so that 5. so as not to
