# Exercises: Either/Neither

Fill in either or neither:

1. ___ of the answers is correct. They're both wrong.
2. ___ option works for me. I'm flexible.
3. I don't like ___ of these colors.
4. ___ she ___ her sister came.
5. Which one? ___ one is fine.

**Answers:** 1. Neither 2. Either 3. either 4. Neither...nor 5. Either
