# Reflexive Pronouns

Reflexive pronouns refer back to the subject.

## Forms

| Subject | Reflexive |
|---------|-----------|
| I | myself |
| you (sg) | yourself |
| he | himself |
| she | herself |
| it | itself |
| we | ourselves |
| you (pl) | yourselves |
| they | themselves |

## Uses

### 1. Same Subject and Object
- *I hurt **myself**.*
- *She taught **herself** to play piano.*
- *He cut **himself** shaving.*

### 2. Emphasis (= alone/personally)
- *I'll do it **myself**.* (without help)
- *The president **himself** came.*
- *Did you make this **yourself**?*

### 3. By + Reflexive = Alone
- *She lives by **herself**.*
- *I prefer to work by **myself**.*

## Common Expressions

- *Help yourself* (to food)
- *Enjoy yourself*
- *Behave yourself*
- *Make yourself at home*

## Common Mistakes

- ✗ *John and myself went...*
- ✓ *John and I went...*

- ✗ *Give it to myself*
- ✓ *Give it to me*
