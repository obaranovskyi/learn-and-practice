# Suffixes for Adverbs (-ly)

Most adverbs are formed by adding **-ly** to adjectives.

## Regular Formation

- quick → quickly
- careful → carefully
- happy → happily

## Spelling Rules

- **-y → -ily**: happy → happily
- **-le → -ly**: gentle → gently, simple → simply
- **-ic → -ically**: basic → basically, automatic → automatically
- **-ful → -fully**: careful → carefully
- **-ll → -lly**: full → fully

## Irregular Adverbs

- good → well (NOT goodly)
- fast → fast (same form)
- hard → hard (hardly = barely)
- late → late (lately = recently)

## Adjectives Ending in -ly

Some -ly words are adjectives:
- friendly, lovely, lonely, silly, ugly

These don't have adverb forms; use "in a ___ way/manner":
- *She spoke in a friendly way.* (NOT friendlily)
