# -ed vs -ing Adjectives

These adjective pairs come from verbs but have different meanings.

## The Rule

- **-ed**: How someone FEELS (passive - receiving the feeling)
- **-ing**: What CAUSES the feeling (active - creating the feeling)

## Examples

| -ed (feeling) | -ing (causing) |
|--------------|----------------|
| *I'm **bored**.* | *The movie is **boring**.* |
| *She's **interested**.* | *The book is **interesting**.* |
| *He was **surprised**.* | *The news was **surprising**.* |
| *We're **tired**.* | *The journey was **tiring**.* |
| *They felt **excited**.* | *The game was **exciting**.* |

## Common Pairs

- amazed / amazing
- annoyed / annoying
- confused / confusing
- depressed / depressing
- disappointed / disappointing
- embarrassed / embarrassing
- exhausted / exhausting
- fascinated / fascinating
- frightened / frightening
- satisfied / satisfying
- shocked / shocking
- worried / worrying

## Common Mistakes

- ✗ *I'm boring.* (= I cause boredom)
- ✓ *I'm bored.* (= I feel boredom)

- ✗ *The movie was bored.*
- ✓ *The movie was boring.*

## Memory Tip

- **-ed** = **E**motion felt by someone
- **-ing** = thing/person **I**nducing the feeling
