# Exercises: Comparative/Superlative Adverbs

Write the comparative and superlative:

1. fast → ___, ___
2. carefully → ___, ___
3. well → ___, ___
4. early → ___, ___
5. badly → ___, ___

**Answers:** 1. faster, fastest 2. more carefully, most carefully 3. better, best 4. earlier, earliest 5. worse, worst
