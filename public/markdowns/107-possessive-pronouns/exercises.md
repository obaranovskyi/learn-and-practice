# Exercises: Possessive Pronouns

Replace the underlined words:

1. This is **my book**. → This is ___.
2. Is that **your car**? → Is that ___?
3. **Their house** is bigger. → ___ is bigger.
4. **Her idea** was better. → ___ was better.
5. I met a friend. It was **a friend of him**. → a friend of ___

**Answers:** 1. mine 2. yours 3. Theirs 4. Hers 5. his
