# Exercises: So/Such...That

Fill in so or such:

1. It was ___ hot that we couldn't sleep.
2. She's ___ a kind person that everyone likes her.
3. He ran ___ fast that he won easily.
4. They were ___ helpful people!
5. There was ___ much noise that I couldn't concentrate.

**Answers:** 1. so 2. such 3. so 4. such 5. so
