# Comparative & Superlative Modifiers

Modifiers strengthen or weaken comparisons.

## Strengthening Comparatives

**much / far / a lot / considerably / significantly + comparative**
- *She's **much taller** than me.*
- *This is **far more expensive**.*
- *It's **a lot better** now.*
- *He's **considerably older**.*

## Weakening Comparatives

**a little / a bit / slightly + comparative**
- *It's **a little colder** today.*
- *She's **a bit taller** than her sister.*
- *This is **slightly more difficult**.*

## With Superlatives

**by far / easily / much + superlative**
- *She's **by far the best** student.*
- *This is **easily the most difficult** test.*
- *It's **much the cheapest** option.*

## Degree Modifiers

| Large difference | Small difference |
|-----------------|------------------|
| much | a little |
| far | a bit |
| a lot | slightly |
| considerably | somewhat |
| significantly | marginally |

## Not with "very"

- ✗ *very better, very more expensive*
- ✓ *much better, much more expensive*

## Special: "Even"

For emphasis/surprise:
- *It's **even better** than I thought!*
- *She's **even more beautiful** in person.*
