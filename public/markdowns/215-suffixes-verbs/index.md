# Suffixes for Verbs

Verb suffixes turn adjectives/nouns into verbs.

## Common Suffixes

### -ify (make)
- simple → simplify
- beautiful → beautify
- class → classify
- pure → purify

### -ize/-ise (make/become)
- modern → modernize
- real → realize
- computer → computerize
- memory → memorize

### -en (make/become)
- short → shorten
- wide → widen
- weak → weaken
- strength → strengthen

## Meanings

- **-ify**: make X → *simplify = make simple*
- **-ize**: make/treat as X → *modernize = make modern*
- **-en**: make/become X → *widen = make wide*

## British vs American

- **-ize** (AmE) / **-ise** (BrE): organize/organise
- Both are acceptable
