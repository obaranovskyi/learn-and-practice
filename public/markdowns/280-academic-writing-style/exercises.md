# Exercises: Academic Style

Make academic: 1. I think this is wrong. 2. This shows that... 3. But on the other hand... 4. So, to sum up... 5. I looked at the data.

**Answers:** 1. It could be argued that this is incorrect. 2. This demonstrates/indicates that... 3. However, / Nevertheless, 4. In conclusion, / To summarize, 5. The data was examined.
