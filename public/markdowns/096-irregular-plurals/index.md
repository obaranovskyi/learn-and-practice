# Irregular Plurals

## Vowel Change

- *man → men*
- *woman → women*
- *foot → feet*
- *tooth → teeth*
- *goose → geese*
- *mouse → mice*

## -en Ending

- *child → children*
- *ox → oxen*

## Same Singular and Plural

- *sheep → sheep*
- *deer → deer*
- *fish → fish* (or fishes for species)
- *aircraft → aircraft*
- *series → series*
- *species → species*

## Latin/Greek Origins

- *analysis → analyses*
- *crisis → crises*
- *thesis → theses*
- *phenomenon → phenomena*
- *criterion → criteria*
- *datum → data*
- *medium → media*
- *cactus → cacti*
- *focus → foci* (or focuses)

## Compound Nouns

Pluralize the main noun:
- *mother-in-law → mothers-in-law*
- *passer-by → passers-by*
- *attorney general → attorneys general*
