# Mixed Conditionals

Mixing elements from 2nd and 3rd conditionals.

## Type 1: Past → Present
Form: If + past perfect, would + base verb

- If I **had studied** medicine, I **would be** a doctor now.
(Past action → present result)

## Type 2: Present → Past
Form: If + past simple, would have + past participle

- If I **were** braver, I **would have asked** her out.
(Present quality → past action)

## Examples
- If I **hadn't moved** here, I **wouldn't know** you. (past→present)
- If she **spoke** French, she **would have got** the job. (present→past)
