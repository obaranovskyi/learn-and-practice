# Exercises: Anticipatory It

Identify "it" type (anticipatory, referential, empty): 1. It is raining. 2. It is important to study. 3. I bought a car. It's red. 4. I find it hard to focus. 5. It's 5 PM.

**Answers:** 1. empty (weather) 2. anticipatory 3. referential 4. anticipatory (object) 5. empty (time)
