# Used to / Would (Past Habits)

## Used to
For past habits AND states no longer true.
- I **used to** smoke. (past habit)
- I **used to** live in Paris. (past state)

## Would
For past habits only (NOT states).
- We **would** play here as kids. ✓
- ❌ I would live in Paris.

## Key Difference

| | Used to | Would |
|---|---------|-------|
| Past habits | ✓ | ✓ |
| Past states | ✓ | ✗ |

## Forms
- **Used to** + base verb
- **Didn't use to** (negative)
- **Did you use to...?** (question)
