# Exercises: Extraposition

Rewrite using extraposition: 1. That she resigned is surprising. 2. To learn a language is difficult. 3. That he lied is obvious. 4. Crying won't help. 5. To be honest is important.

**Answers:** 1. It's surprising that she resigned. 2. It's difficult to learn a language. 3. It's obvious that he lied. 4. It's no use crying. 5. It's important to be honest.
