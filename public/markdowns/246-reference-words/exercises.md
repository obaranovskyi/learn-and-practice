# Exercises: Reference Words

Choose: this, that, these, those, one: 1. ___ is an interesting point. (just made) 2. I need a book. Do you have ___? 3. ___ problems are serious. (mentioned earlier) 4. I met Tom and Mary. The former is tall; ___ latter is short.

**Answers:** 1. This 2. one 3. Those 4. the
