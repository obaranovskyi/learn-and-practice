# Exercises: First Conditional

## Exercise 1: Complete the Sentences

1. If it ______ (rain), we ______ (stay) inside.
2. If you ______ (not/hurry), you ______ (miss) the train.
3. She ______ (be) happy if she ______ (get) the job.
4. If I ______ (have) time, I ______ (help) you.
5. If they ______ (not/study), they ______ (fail) the test.

**Answers:**
1. rains, will stay
2. don't hurry, will miss
3. will be, gets
4. have, will help
5. don't study, will fail

---

## Exercise 2: Match the Halves

| If clause | Main clause |
|-----------|-------------|
| 1. If you eat too much, | a. I'll lend you mine. |
| 2. If you need a pen, | b. you'll feel sick. |
| 3. If the bus doesn't come, | c. will you help me? |
| 4. If you have time, | d. we'll take a taxi. |

**Answers:**
1-b, 2-a, 3-d, 4-c

## Exercise 3: Rewrite Using Unless

1. If you don't leave now, you'll be late.
2. If it doesn't stop raining, we won't go out.
3. I won't pass if I don't study.
4. If she doesn't call, I won't know.

**Answers:**
1. Unless you leave now, you'll be late.
2. Unless it stops raining, we won't go out.
3. I won't pass unless I study.
4. Unless she calls, I won't know.

---

## Exercise 4: Correct the Mistakes

1. If it will rain, I will stay home.
2. If you will study hard, you will pass.
3. I go to the beach if the weather is nice.
4. If you won't help me, I'll be angry.
5. Unless you don't hurry, you'll miss it.

**Answers:**
1. If it **rains**, I will stay home.
2. If you **study** hard, you will pass.
3. I **will** go to the beach if the weather is nice.
4. If you **don't** help me, I'll be angry.
5. Unless you hurry, you'll miss it. *(Unless already means "if not")*

## Exercise 5: Make First Conditional Sentences

Create sentences using the prompts.

1. you / be late / boss / be angry
2. weather / be nice / we / have a picnic
3. she / not call / I / not know
4. you / help me / I / finish faster
5. it / snow / schools / close

**Answers:**
1. If you are late, your boss will be angry.
2. If the weather is nice, we will have a picnic.
3. If she doesn't call, I won't know.
4. If you help me, I will finish faster.
5. If it snows, schools will close.
