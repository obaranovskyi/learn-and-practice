# If, Unless, In case (Condition)

These conjunctions express conditions.

## If

Basic condition:
- ***If** it rains, we'll stay home.*
- *I'll help **if** you ask.*
- ***If** I were you, I'd accept.*

## Unless

**Meaning**: if...not, except if
- *I'll go **unless** it rains.* (= if it doesn't rain)
- ***Unless** you hurry, you'll miss it.*
- *Don't call **unless** it's urgent.*

**Note**: Don't use negative with unless
- ✗ *Unless you don't hurry...*
- ✓ *Unless you hurry...*

## In case

**Meaning**: because something might happen (precaution)
- *Take an umbrella **in case** it rains.* (it might rain)
- *I'll leave a note **in case** she comes.*

**Unlike if**: Action happens regardless
- *If it rains, take an umbrella.* (only if it rains)
- *Take an umbrella in case it rains.* (take it now, as precaution)

## Provided (that) / As long as

**Meaning**: on condition that
- *You can go **provided** you're back by 10.*
- ***As long as** you try, you'll improve.*

## Comparison

| Word | Meaning | Example |
|------|---------|---------|
| if | condition | If it rains, stay home. |
| unless | if not | Unless it rains, go out. |
| in case | precaution | Take an umbrella in case it rains. |
| provided | on condition | Go provided you return early. |
