# Passive with Modals

Form: **modal + be + past participle**

## Examples

| Active | Passive |
|--------|---------|
| They can fix it | It **can be fixed** |
| You must complete it | It **must be completed** |
| She should do it | It **should be done** |
| They might cancel it | It **might be cancelled** |
| We have to finish it | It **has to be finished** |

## Past Modal Passive
Form: modal + have been + past participle

- It **could have been avoided**.
- This **should have been done** earlier.
- It **must have been stolen**.
