# Plural Formation (Regular)

## Basic Rule: Add -s

- *book → books*
- *car → cars*
- *table → tables*

## Add -es

After **s, ss, sh, ch, x, z**:
- *bus → buses*
- *class → classes*
- *dish → dishes*
- *watch → watches*
- *box → boxes*

## Words ending in -y

**Consonant + y**: Change y to i, add -es
- *baby → babies*
- *city → cities*
- *party → parties*

**Vowel + y**: Just add -s
- *boy → boys*
- *key → keys*
- *day → days*

## Words ending in -o

**Most**: Add -es
- *potato → potatoes*
- *tomato → tomatoes*
- *hero → heroes*

**Music/abbreviations**: Just -s
- *piano → pianos*
- *photo → photos*
- *video → videos*

## Words ending in -f/-fe

Change to -ves (usually):
- *leaf → leaves*
- *knife → knives*
- *wife → wives*
- *life → lives*

Exceptions (just -s): *roof → roofs*, *chief → chiefs*
