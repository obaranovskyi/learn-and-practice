# Abstract vs Concrete Nouns

## Concrete Nouns

Things you can perceive with your senses (see, touch, hear, smell, taste):

- **Objects**: book, table, car, phone
- **People**: teacher, child, doctor
- **Animals**: dog, elephant, bird
- **Places**: school, Paris, beach
- **Substances**: water, gold, air

*You can see the **book** on the **table**.*

## Abstract Nouns

Ideas, concepts, feelings - cannot be perceived with senses:

| Category | Examples |
|----------|----------|
| Emotions | love, fear, happiness, anger |
| Qualities | beauty, courage, honesty, patience |
| Concepts | freedom, justice, time, knowledge |
| States | childhood, health, poverty, death |
| Events | birthday, marriage, meeting |

*Her **courage** inspired **hope**.*

## Forming Abstract Nouns

| From Adjective | Abstract Noun |
|----------------|---------------|
| happy | happiness |
| kind | kindness |
| brave | bravery |
| free | freedom |
| true | truth |

| From Verb | Abstract Noun |
|-----------|---------------|
| grow | growth |
| arrive | arrival |
| decide | decision |
| move | movement |
