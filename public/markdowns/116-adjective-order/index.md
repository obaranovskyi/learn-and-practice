# Adjective Order

When using multiple adjectives, they follow a specific order.

## The Order: OSASCOMP

1. **O**pinion - lovely, beautiful, horrible
2. **S**ize - big, small, tiny, huge
3. **A**ge - old, new, young, ancient
4. **S**hape - round, square, flat
5. **C**olor - red, blue, green
6. **O**rigin - French, Japanese, American
7. **M**aterial - wooden, silk, metal
8. **P**urpose - sleeping (bag), cooking (pot)

## Examples

- *a lovely small old house* (opinion + size + age)
- *a big round wooden table* (size + shape + material)
- *an expensive new Italian car* (opinion + age + origin)
- *a beautiful long white silk dress* (opinion + size + color + material)

## Quick Memory Aid

**OSASCOMP** or remember: **"Old Sam Always Sits Counting Old Money Patiently"**

## Common Patterns

- Opinion usually comes first
- Purpose adjectives come last (before the noun)
- Material comes before purpose
- We rarely use more than 3 adjectives

## Notes

- Use commas between similar category adjectives: *a cold, windy day*
- No commas between different categories: *a beautiful old house*
