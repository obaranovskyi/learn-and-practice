# Exercises: Adjective Order

Put the adjectives in the correct order:

1. a ___ dress (silk / blue / beautiful)
2. a ___ house (old / lovely / stone)
3. ___ shoes (Italian / black / leather)
4. a ___ car (sports / red / new)
5. a ___ table (wooden / round / big)

**Answers:** 1. beautiful blue silk 2. lovely old stone 3. black Italian leather 4. new red sports 5. big round wooden
