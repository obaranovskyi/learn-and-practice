# Would rather / Would sooner

## Own preference: would rather + base verb
- I**'d rather stay** home.
- She**'d rather not go**.

## Preference about someone else: would rather + past simple
- I**'d rather you didn't** smoke.
- I**'d rather he came** tomorrow.

## Would sooner (same meaning, less common)
- I'd sooner **die** than apologize.

## Prefer vs Would rather
- **Prefer** = general preference
- **Would rather** = specific situation
