# Informal Conversational Patterns

Natural speech patterns among friends and casual acquaintances.

## Greetings

- *Hey! / Hi! / What's up?*
- *How's it going? / How are things?*
- *Long time no see!*
- *What's new?*

## Starting Topics

- *Guess what!* (exciting news)
- *You know what?* (introducing topic)
- *So, anyway...* (changing topic)
- *Speaking of which...* (related topic)

## Showing Interest

- *No way! / Seriously? / Really?*
- *That's crazy! / That's awesome!*
- *Oh my god! / OMG!*
- *Tell me more!*

## Agreeing

- *Totally! / Absolutely!*
- *I know, right?*
- *Same here! / Me too!*
- *You're telling me!*
- *Exactly! / Precisely!*

## Disagreeing (softly)

- *I don't know about that...*
- *Not really... / Not exactly...*
- *I'm not so sure...*
- *Yeah, but...*

## Ending Conversations

- *Anyway, I gotta go.*
- *Well, I should get going.*
- *Catch you later! / See ya!*
- *Let's catch up soon!*
- *Take care! / Later!*

## Features

- Contractions everywhere
- Incomplete sentences
- Fillers (um, like, you know)
- Slang and idioms
- Fast-paced back-and-forth
